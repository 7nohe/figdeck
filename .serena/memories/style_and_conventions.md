# Style and conventions
- Language: TypeScript (ES module in CLI); plugin code targets es2018; UTF-8.
- Lint/format: Biome enabled with recommended rules; 2-space indent, space style, double quotes, semicolons always.
- Naming/types: TypeScript types/interfaces in src/types.ts; keep CLI commands defined via commander; shared SlideContent shapes consistent with docs.
- Docs: Japanese-language README/docs; maintain concise headings and examples.
- Avoid non-ASCII unless needed; follow existing markdown examples for user-facing text.
