# Callouts

Showcase of callout directive syntax

---

## NOTE Callout

Helpful information for users.

:::note
This is a NOTE callout. Use it for helpful information that users should know.
:::

---

## TIP Callout

Best practices and recommendations.

:::tip
This is a TIP callout. Use it to share best practices or recommendations.
:::

---

## WARNING Callout

Potential issues or deprecated features.

:::warning
This is a WARNING callout. Use it to warn about potential issues or deprecated features.
:::

---

## CAUTION Callout

Destructive actions or security concerns.

:::caution
This is a CAUTION callout. Use it for destructive actions or security-sensitive operations.
:::

---

## Rich Formatting

Callouts support inline formatting.

:::tip
Use `--secret` for **authentication** when using `--allow-remote`. See the [documentation](https://figdeck.7nohe.dev) for details.
:::

---

## Callout Colors

Each callout type has a distinct accent color:

- **NOTE**: Blue
- **TIP**: Green
- **WARNING**: Orange
- **CAUTION**: Red

---

## Callouts in Columns

Callouts work inside multi-column layouts.

:::columns
:::column
:::note
A note in the left column.
:::
:::column
:::tip
A tip in the right column.
:::
:::
