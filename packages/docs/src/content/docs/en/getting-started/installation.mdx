## Prerequisites

Before installing figdeck, make sure you have the following:

### Figma Desktop App

<Aside type="caution">
figdeck only works with the **Figma Desktop App**. The browser version of Figma does not support WebSocket connections required by the plugin.
</Aside>

Download from: [figma.com/downloads](https://www.figma.com/downloads/)

### Node.js

Node.js 20 or later is required to run the CLI.

Download from: [nodejs.org](https://nodejs.org/)

You can verify your installation by running:

```bash
node --version
```

## Figma Plugin

Install the figdeck plugin from Figma Community:

**[figdeck - Figma Community](https://www.figma.com/community/plugin/1577342026252824260/figdeck)**

## CLI

You can use figdeck directly with `npx`, `pnpm dlx`, or `bunx`, or install it globally.

### Using npx, pnpm dlx, or bunx (Recommended)

<Tabs syncKey="pkg">
	<TabItem label="npx">

```bash
npx figdeck@latest init
npx figdeck@latest serve slides.md
```

	</TabItem>
	<TabItem label="pnpm dlx">

```bash
pnpm dlx figdeck@latest init
pnpm dlx figdeck@latest serve slides.md
```

	</TabItem>
	<TabItem label="bunx">

```bash
bunx figdeck@latest init
bunx figdeck@latest serve slides.md
```

	</TabItem>
</Tabs>

### Global Installation

```bash
npm install -g figdeck
```
