figdeck is a tool that automatically generates Figma Slides from Markdown files. The CLI parses Markdown and communicates with the Figma Plugin via WebSocket to create slides.

## Features

<CardGrid stagger>
	<Card title="Markdown to Slides" icon="document">
		Write your slides in simple Markdown. No need to learn complex tools or design software.
	</Card>
	<Card title="Live Preview" icon="rocket">
		Changes are reflected instantly in Figma via WebSocket. See your slides update in real-time as you edit.
	</Card>
	<Card title="Figma Integration" icon="puzzle">
		Uses native Figma Slides features. Leverage the full power of Figma's design capabilities.
	</Card>
	<Card title="Customizable" icon="setting">
		Control styling via frontmatter and custom themes. Make your slides match your brand.
	</Card>
	<Card title="Developer Friendly" icon="laptop">
		Integrates with your existing Git workflow. Version control your slides like code.
	</Card>
</CardGrid>

## How it works

<CardGrid>
	<Card title="1. Write Markdown" icon="pencil">
		Create a `slides.md` file using the supported syntax.
	</Card>
	<Card title="2. Run CLI" icon="rocket">
		Start the `figdeck serve` command to watch for changes.
	</Card>
	<Card title="3. Connect Plugin" icon="puzzle">
		Open the figdeck plugin in Figma and connect.
	</Card>
	<Card title="4. Generate" icon="approve-check">
		Slides are generated and updated in real-time.
	</Card>
</CardGrid>
