## 事前準備

figdeck をインストールする前に、以下の要件を確認してください：

### Figma デスクトップアプリ

<Aside type="caution">
figdeck は **Figma デスクトップアプリ** でのみ動作します。ブラウザ版の Figma は、プラグインが必要とする WebSocket 接続をサポートしていません。
</Aside>

ダウンロード: [figma.com/downloads](https://www.figma.com/downloads/)

### Node.js

CLI を実行するには Node.js 20 以降が必要です。

ダウンロード: [nodejs.org](https://nodejs.org/)

インストールを確認するには、以下を実行してください：

```bash
node --version
```

## Figma Plugin

Figma Community から figdeck プラグインをインストールしてください：

**[figdeck - Figma Community](https://www.figma.com/community/plugin/1577342026252824260/figdeck)**

## CLI

figdeck は `npx`、`pnpm dlx`、または `bunx` で直接実行するか、グローバルにインストールして使用できます。

### npx、pnpm dlx、または bunx を使用する（推奨）

<Tabs syncKey="pkg">
	<TabItem label="npx">

```bash
npx figdeck@latest init
npx figdeck@latest serve slides.md
```

	</TabItem>
	<TabItem label="pnpm dlx">

```bash
pnpm dlx figdeck@latest init
pnpm dlx figdeck@latest serve slides.md
```

	</TabItem>
	<TabItem label="bunx">

```bash
bunx figdeck@latest init
bunx figdeck@latest serve slides.md
```

	</TabItem>
</Tabs>

### グローバルインストール

```bash
npm install -g figdeck
```
