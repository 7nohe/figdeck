figdeck は Markdown ファイルから Figma Slides を自動生成するツールです。CLI で Markdown をパースし、Figma Plugin と WebSocket で連携してスライドを作成します。

## 特徴

<CardGrid stagger>
	<Card title="Markdown to Slides" icon="document">
		シンプルな Markdown でスライドを作成できます。複雑なツールやデザインソフトを学ぶ必要はありません。
	</Card>
	<Card title="ライブプレビュー" icon="rocket">
		WebSocket を通じて変更が即座に Figma に反映されます。編集しながらリアルタイムでスライドの更新を確認できます。
	</Card>
	<Card title="Figma 統合" icon="puzzle">
		Figma Slides のネイティブ機能を使用します。Figma のデザイン機能をフルに活用できます。
	</Card>
	<Card title="カスタマイズ可能" icon="setting">
		フロントマターやカスタムテーマでスタイルを制御できます。ブランドに合わせてスライドをカスタマイズできます。
	</Card>
	<Card title="開発者フレンドリー" icon="laptop">
		既存の Git ワークフローと統合しやすくなっています。コードのようにスライドをバージョン管理できます。
	</Card>
</CardGrid>

## 仕組み

<CardGrid>
	<Card title="1. Markdown を書く" icon="pencil">
		サポートされている記法を使って `slides.md` を作成します。
	</Card>
	<Card title="2. CLI を実行" icon="rocket">
		`figdeck serve` コマンドを実行して変更を監視します。
	</Card>
	<Card title="3. Plugin を接続" icon="puzzle">
		Figma で figdeck プラグインを開き、接続します。
	</Card>
	<Card title="4. 生成" icon="approve-check">
		スライドがリアルタイムで生成・更新されます。
	</Card>
</CardGrid>
